Mafia Host - client-only web app (v2)
Files:
- index.html
- style.css
- script.js

How to use:
- Open index.html in browser (works offline).
- Home: Two buttons - New Game and Continue Game.
- New Game: set passkey, number of players, check characters and counts (sum must equal players).
- Assign players: pass device to each player, they enter name, click OK to reveal role.
- After assigning all players, enter passkey to open dashboard.
- Dashboard: manage votes, toggles. Use Next Round to reset toggles (except Dead/Alive) and votes.
- Game auto-saves to localStorage; Continue Game resumes saved game.
- Partial form selections auto-save while initiating a new game.

This package is intended to be deployed as static files (e.g., GitHub Pages).
